ascendingBy

  • ascendingBy<T>(...getters: function[]): function
  • ascendingBy<T, K>(...keys: K[]): function
    • Defined in array/sort.ts:39

    Sorts array in ascending order by values provided from callbacks. First callback has highest priority in sorting and so on. It accepts as many callbacks as You need.

    example

    names.sort(ascendingBy( x => x.name, x => x.lastName, x => x.age )) // [ // { name: 'andrew', lastName: 'Aa', age: 1 }, // { name: 'andrew', lastName: 'Bb', age: 1 }, // { name: 'andrew', lastName: 'Bb', age: 2 }, // { name: 'billy', lastName: 'Cc', age: 1 }, // { name: 'billy', lastName: 'Cc', age: 5 }, // ]
